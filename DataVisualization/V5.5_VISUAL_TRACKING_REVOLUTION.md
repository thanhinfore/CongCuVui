## üìê Easing Functions Deep Dive

### easeInOutQuart (Main)
```javascript
t < 0.5 ? 8 * t^4 : 1 - 8 * (1 - t)^4
```

- **Acceleration**: Moderate
- **Deceleration**: Moderate
- **Feel**: Professional, balanced

### easeInOutQuint (Y-axis positions)
```javascript
easeInOutQuint(t) {
    return t < 0.5
        ? 16 * t * t * t * t * t

```
- **Acceleration**: Very slow start
- **Deceleration**: Very slow end
- **Feel**: Ultra-smooth, easy to track
- **Power**: 5 (quintic)

```javascript
    const r = parseInt(hex.substr(0, 2), 16);
```
